# 🚀 Full Stack Project

This is a full stack web application with a Python backend and a React (Vite) frontend.

---

## 🛠️ Setup Instructions

### 📦 Backend (Python)

```bash
cd backend
pip install -r requirements.txt
```

---

### 💻 Frontend (React + Vite)

```bash
cd frontend
npm install
npm run dev
```
